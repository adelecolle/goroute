Este documento apresenta a arquitetura técnica completa da plataforma digital Go Route — do site institucional ao portal de motoristas, sistema de reservas, rastreamento em tempo real e estratégia de marketing. A implantação é progressiva em 6 fases, começando pelo essencial e expandindo conforme o negócio cresce.

Diagrama 02 · 06

As camadas
da plataforma

O sistema funciona em camadas — cada uma com uma responsabilidade clara. Os dados fluem de cima para baixo e as respostas sobem de volta.

01
Usuário — Navegador & Celular
Clientes, motoristas e Diow & Leandro acessam tudo pelo navegador. Sem instalar app. O PWA permite "adicionar à tela inicial" como se fosse um app nativo.
Chrome · Safari PWA instalável WhatsApp E-mail
↓
Requisições HTTP · WebSocket
02
Frontend — Interface Visual
O que o usuário vê e clica. Desenvolvido em Next.js (código real — não Wix ou Squarespace). Hospedado na Vercel com deploy automático a cada atualização. Cloudflare protege e acelera.
Next.js React Vercel Cloudflare CDN Sanity CMS Google Maps embed Tidio Chatbot
↓
API REST · Realtime WebSocket
03
Backend — Cérebro da Operação
Onde os dados vivem e as regras de negócio acontecem. Reservas, usuários, motoristas, pagamentos e GPS — tudo armazenado e processado no Supabase (PostgreSQL gerenciado).
Supabase PostgreSQL Supabase Auth Supabase Realtime (GPS) Supabase Storage Edge Functions
↓
Webhooks · Chamadas de API externas
04
Serviços Externos — Pagamentos · Comunicações · APIs
Serviços especializados que o backend chama conforme necessário. Cada um faz uma coisa muito bem.
Mercado Pago · Pix · Cartão Z-API WhatsApp Resend E-mail AviationStack · Voos OpenWeatherMap Google Maps API
↓
Rastreamento · Analytics · Indexação
05
Marketing & Visibilidade
Ferramentas invisíveis para o usuário, mas essenciais para o crescimento orgânico da Go Route. Trazem tráfego, medem conversões e aumentam a presença no Google e Instagram.
Google Analytics 4 Search Console Google Business Profile Meta Pixel Instagram Feed
Diagrama 03 · 06

Do clique
à confirmação

O fluxo completo de uma reserva — do momento em que o cliente encontra a Go Route no Google até a confirmação chegar no WhatsApp.

01

Cliente encontra a Go Route

Pesquisa "transfer gramado aeroporto" no Google → site aparece nos resultados orgânicos + Google Business Profile → acessa pelo celular.

Google Search · SEO Next.js · Google Business
02

Preenche o formulário de reserva

Seleciona rota, data e veículo. Informa o número do voo → o sistema consulta a AviationStack API e preenche automaticamente o horário real. Se o voo atrasar depois, o sistema já sabe.

Next.js · AviationStack API
03

Pagamento via Pix ou Cartão

Checkout com resumo da reserva. Pix: confirmação em segundos. Cartão: processado pelo Mercado Pago (3,49%). O valor entra direto na conta PJ da Go Route.

Mercado Pago · Pix 0% · Cartão 3,49%
04

Mercado Pago notifica o sistema

Após confirmação do pagamento, o Mercado Pago envia um webhook automático para o Supabase — "reserva paga". O sistema registra e dispara as próximas ações sem intervenção humana.

Webhook Mercado Pago → Supabase PostgreSQL
05

Confirmação automática para o cliente

Simultaneamente: WhatsApp com detalhes completos + e-mail HTML com resumo da viagem, endereço de embarque e previsão do tempo para o dia em Gramado.

Z-API WhatsApp · Resend E-mail · OpenWeatherMap
06

Motorista recebe a corrida

Sistema notifica o motorista disponível por WhatsApp e pelo portal. Ele tem 30 minutos para aceitar. Se recusar, vai automaticamente para o próximo da lista.

Portal motorista · Z-API · Supabase
07

Lembrete automático 24h antes

No dia anterior, o sistema dispara um lembrete por WhatsApp com todos os detalhes e contato do motorista. Sem que Diow ou Leandro precisem fazer nada.

Cron Job Supabase · Z-API
Diagrama 04 · 06

O dia
da viagem

Do motorista partir para buscar o cliente até o pedido de avaliação 2 horas após a chegada em Gramado.

01

Motorista abre o portal e parte

Acessa o PWA pelo celular (sem instalar app — abre no navegador). Vê detalhes da corrida: nome do cliente, terminal, placa de identificação. Aperta "Iniciar viagem".

PWA Next.js · Supabase Auth
02

GPS começa a transmitir

O celular do motorista passa a enviar a posição GPS a cada 15–30 segundos para o Supabase Realtime — um canal de transmissão ao vivo. Qualquer navegador conectado recebe a posição instantaneamente.

Geolocation API · Supabase Realtime WebSocket
03

Cliente recebe link de rastreamento

O sistema envia automaticamente por WhatsApp um link público — como goroutetransfer.com.br/track/ABC123. Qualquer pessoa com o link vê o mapa. Sem criar conta, sem instalar nada.

Z-API · Link público sem autenticação
04

Mapa ao vivo no celular do cliente

O carro se move no mapa em tempo real. ETA atualizado continuamente. A posição vai do Supabase diretamente ao Google Maps via WebSocket — sem precisar atualizar a página. A família pode acompanhar também.

Google Maps JS API · Supabase Realtime
05

Motorista conclui a viagem

Aperta "Concluir viagem" no portal. O sistema registra horário de chegada e calcula o repasse automaticamente conforme a categoria do veículo (ex: 78% para o motorista na categoria Executive).

Portal motorista · Lógica de repasse · Supabase
06

Pedido de avaliação automático

2 horas após a conclusão, o sistema envia WhatsApp pedindo avaliação no Google Maps. Crítico para o crescimento orgânico: avaliações no Google são o principal fator de rankeamento local e são citadas diretamente por IAs como ChatGPT e Gemini.

Z-API · Google Business Profile
Diagrama 05 · 06

Tech stack
completo

Cada ferramenta, o que faz exatamente, por que foi escolhida para a Go Route e quanto custa por mês.

Frontend
Next.js + React
O framework que constrói o site. Gera as páginas, cuida de roteamento, performance e SEO automaticamente. Não é Wix — é código que a Go Route controla 100%, sem plataforma intermediária.
Gratuito — open source
Hospedagem Frontend
Vercel
Hospeda o site Next.js com deploy automático a cada atualização (push no GitHub → site atualizado em 30 segundos). CDN global, HTTPS automático. Feito pela mesma empresa do Next.js.
Gratuito no início
CDN · DNS · Segurança
Cloudflare
Fica na frente de tudo. Protege contra ataques DDoS, acelera o carregamento globalmente e gerencia o DNS — direciona goroutetransfer.com.br para a Vercel.
Gratuito
CMS · Edição de Conteúdo
Sanity.io
Painel onde Diow e Leandro editam textos, fotos, eventos da Serra Gaúcha e artigos do blog — sem tocar em código. O site busca esse conteúdo via API quando carrega.
Gratuito no plano básico
Backend · Banco de Dados
Supabase
O coração da plataforma. PostgreSQL + autenticação + storage + realtime em um só serviço. Guarda todas as reservas, usuários, motoristas, histórico de GPS e documentos.
R$ 0 grátis → R$ 100/mês Pro
Pagamentos
Mercado Pago
Processa Pix (0% de taxa) e cartão (3,49%). Quando o pagamento é confirmado, envia webhook automático para o Supabase. Zero mensalidade — Go Route só paga sobre o que vende.
0% Pix · 3,49% cartão · sem mensalidade
WhatsApp Automatizado
Z-API
Conecta a plataforma ao WhatsApp. Envia confirmações, lembretes, link de rastreamento GPS e pedido de avaliação — tudo automaticamente, sem Diow ou Leandro precisarem digitar nada.
R$ 99–149/mês
E-mail Transacional
Resend.com
Envia e-mails bonitos em HTML de forma programática. Confirmação de reserva com detalhes completos, lembrete de viagem e solicitação de avaliação pós-viagem. Alta entregabilidade.
Gratuito até 3.000/mês
API de Voos
AviationStack
Quando o cliente informa o número do voo (ex: LA3042), essa API retorna horário real, terminal e status. Se o voo atrasar, a plataforma Go Route já sabe e ajusta a corrida automaticamente.
Gratuito até 500 req/mês
Previsão do Tempo
OpenWeatherMap
Busca a previsão para o destino no dia da viagem. Exibida na confirmação de reserva e nas páginas de Gramado, Canela e Bento Gonçalves. "Gramado estará a 8°C — leve casaco."
Gratuito até 1.000 req/dia
GPS · Mapas
Google Maps API
Renderiza o mapa de rastreamento ao vivo. O Supabase Realtime entrega a posição do motorista, o Maps renderiza o ícone se movendo em tempo real e calcula o ETA dinâmico.
Gratuito no free tier inicial
Chatbot
Tidio · Fase 4
Chatbot no canto do site. Fase 4: fluxo de perguntas simples que qualifica e redireciona para WhatsApp. Fase 6: substituído por chatbot com IA (Claude/GPT-4o) para responder autonomamente.
Gratuito no plano básico
Analytics
Google Analytics 4
Rastreia tudo no site: origem dos visitantes, páginas acessadas, tempo de permanência, taxa de conversão em reservas. Dados históricos são valiosos — instalar no dia 1 é obrigatório.
Gratuito
Remarketing
Meta Pixel
Código invisível que identifica visitantes do site. Permite mostrar anúncios no Instagram e Facebook para quem visitou mas não reservou. Instalar agora = coletar dados desde o início.
Gratuito
SEO Local
Google Business Profile
O card da Go Route no Google Maps. Avaliações aqui são citadas por IAs (ChatGPT, Gemini, Perplexity). Verificação leva 5–14 dias — criar e verificar é prioridade zero, antes de qualquer código.
Gratuito
